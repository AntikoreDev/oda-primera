Alfabeto

El alfabeto es el siguiente, y las letras tienen sus propios nombres.

Letter Name Letter Name Letter Name
A A B Be C Ce
D De E E F Efa
G Ge H Hetsa I I
J Je K Ka L Ela
M Ema N Ena O O
P Pe Q Qua R Ra
S Se T Ta U U
V Uva W Biuva X Xe
Y Ye Z Zeta

Aparte las combinaciones de letras se llaman de la siguiente manera

Letter Name Letter Name Letter Name
TS Tsa NN Nne RR Erra
KS Eksa
